Issue Priority URLs % Total Description How to Fix
Response Codes: Rate Limited (HTTP 429)
 Critical 888 30.5% Shopify is returning HTTP 429 Too Many Requests for 888 internal URLs — predominantly collection pages and product pages. These are NOT broken links. The server is throttling the crawler, which means Googlebot may also be rate-limited, preventing proper indexation of key fabric category pages. In Screaming Frog: Configuration → Speed → reduce to 1–2 requests/sec. Add Crawl-delay: 2 to robots.txt. Check Google Search Console → Settings → Crawl Stats to verify Googlebot is not also being throttled. Contact Shopify support if crawl budget issues persist.
Images: Missing Alt Text
 Critical 297 10.2% 297 product images have an alt attribute present but the value is empty. Google Image Search is one of the highest-intent discovery channels for fabric buyers. Empty alt attributes are an accessibility failure (WCAG 2.1 AA) and prevent any image search ranking for these products. In Shopify Admin → Products → click each product image → click Edit alt text. Use format: "[Fabric Name] [Colour/Design] Fabric — Buy Online India | Saroj Fabrics". For 297 images, use Shopify's CSV bulk export to update at scale via a spreadsheet.
H1: Missing on Collection & Static Pages
 Critical 64 2.2% 64 pages — primarily Shopify collection pages and static pages (About, Contact, FAQs, Policies) — have no H1 tag. The H1 is the primary on-page ranking signal that tells Google the topic of the page. Missing H1s weaken collection page topical relevance and ability to rank for fabric category keywords. In Shopify Admin: Themes → Customize → Collection template → verify the collection title renders as <h1> (not <h2> or <div>). For static pages: edit page content and add H1 as the first heading. Format: "[Fabric Type] — Buy [Category] Online India | Saroj Fabrics".
Meta Descriptions: Missing
 Critical 24 0.8% 24 collection pages have no meta description, causing Google to auto-generate snippets from page content — typically uninformative for fabric stores. Meta descriptions directly influence click-through rates from SERPs. For a Shopify fabric store, compelling descriptions that communicate fabric type, availability, and USPs are critical to converting impressions. Shopify Admin → Collections → each collection → SEO section → edit "Description" field. Target 150–160 characters. Include: primary keyword (fabric type), 1–2 USPs (e.g. "wholesale available", "pan-India delivery"), and a soft CTA. Example: "Shop premium embroidery fabrics online at Saroj Fabrics. Hand & machine embroidered silk, georgette, chanderi & more. Bulk orders available."
Images: Missing Alt Attribute (Sitewide Banner)
 High 1 — One CDN image (referenced across 264 pages) is missing the alt attribute entirely from its <img> tag — not just an empty value. This is likely a sitewide hero/banner image embedded in the Shopify theme. A missing alt attribute violates WCAG standards and is distinct from an empty alt value. Locate the image in your theme Liquid files by searching for the filename: "image_24_2x_421a1bd7". Add an alt attribute to the <img> tag: alt="{{ section.settings.image_alt | default: 'Premium Indian Fabrics — Saroj Fabrics' }}". This one fix covers 264 pages simultaneously.

Page Titles: Duplicate
 High 24 0.8% 24 collection pages share duplicate title tags across fabric categories — e.g. "Buy Printed Fabrics Online | Saroj Fabrics" is used for 5 separate collections (Crepe Printed, Chanderi Printed, Muslin Printed, Cotton Printed, Silk Printed). Always include the specific fabric material: "Buy Crepe Printed Fabric Online | Saroj Fabrics" vs. "Buy Chanderi Printed Fabric Online | Saroj Fabrics". Max 60 characters. Use the fabric's most-searched variant name.
Directives: Noindex Pages
 High 2 0.07% 2 product pages with URL slugs containing "mws_apo_1_g2" are marked noindex,nofollow. The slug pattern strongly indicates these are test or staging products from an app installation that were never deleted. They consume crawl budget and represent potential content bloat. In Shopify Admin → Products, search for "mws_apo_1_g2". If these are test products, delete them entirely. If they are real products that were incorrectly noindexed, update the slug to a clean format and remove the noindex directive through the Shopify SEO settings or an SEO app.

For the /collections/all canonical: verify in Shopify Admin whether you want this page indexed. If yes, ensure it has a self-referencing canonical. The remaining 4 (sort params, variants, pagination) require no action — this is standard Shopify canonical behaviour and is correct.

04 — Opportunities

Quick Wins & Optimisation Opportunities

Opportunities do not affect the health score but represent low-effort, high-impact improvements that can lift organic CTR, on-page relevance, and conversion rate for saroj.in. Prioritise the Medium items first — most are Shopify-native fixes requiring no developer work.

OPPORTUNITY

Medium Priority — SEO & Commerce Impact

7 items
Opportunity Scope Recommendation
Product Schema Markup
 All products Add JSON-LD Product schema to all product pages including: name, image, description, brand, offers (price, currency, availability), and aggregateRating. This enables Google Shopping rich results and price snippets — high value for a fabric e-commerce store. Use a Shopify SEO app or add to product.liquid template.
Breadcrumb Schema
 Collection + Product pages Implement BreadcrumbList schema on all collection and product pages. Shopify has a native breadcrumb Liquid snippet — ensure it outputs valid JSON-LD. Breadcrumbs in SERPs increase click-through rate by showing page hierarchy (e.g. Home > Silk Fabrics > Banarasi Brocade).
Open Graph & Twitter Card Tags
 All pages Add og:title, og:description, og:image, and og:type to all pages. Shopify's default theme typically includes these but verify they're populated with unique values per collection. Critical for WhatsApp sharing (primary channel for Indian fabric buyers) — a missing OG image results in no preview thumbnail when links are shared.
Collection Page Content Blocks
 64 collection pages Add 100–200 word descriptive content blocks below the product grid on each collection page. Include primary keyword, fabric characteristics, and use cases. This gives Google topical context beyond just product titles and significantly improves ranking for long-tail fabric queries. Can be added via Shopify's collection description field.
Internal Linking — Cross-Collection Links
 All collections Add "You may also like" or "Related Collections" internal links from each fabric collection to related category pages (e.g. from Chanderi Printed → Chanderi Embroidery → Chanderi Plain). This distributes PageRank across the collection hierarchy and helps Google understand the site's topical clusters.
Blog Content Expansion
 saroj.in/blogs/news The blog section currently has minimal content. Publishing 10–15 fabric-related articles (e.g. "How to choose the right embroidery fabric", "Difference between Chanderi and Banarasi fabrics") creates topical authority, drives informational search traffic, and builds internal links to collection pages. Target 1,000–1,500 words per post.
Page Title Length Optimisation
 All collection pages Review all collection page titles to ensure they are between 40–60 characters. Many currently share a generic format without the fabric material differentiated. After fixing duplicates (see Warnings), also check pixel width stays under 561px using a SERP preview tool such as Mangools or Google's Rich Results Test.

Low Priority — Technical Polish

5 items
Opportunity Scope Recommendation
Image WebP Conversion
 All product images Shopify automatically serves WebP to supported browsers, but verify via Chrome DevTools Network tab that images are being served as WebP, not JPG/PNG. If not, check your theme's image rendering Liquid tags use the {{ image | image_url: width: 800 }} format. This improves Core Web Vitals LCP scores.
Sitemap Optimisation
 saroj.in/sitemap.xml Shopify auto-generates a sitemap. Verify it is submitted in Google Search Console and that all key collection pages are included. Check that test/noindex product URLs are excluded. If the sitemap includes 429-returning URLs, this may reduce Google's trust in sitemap accuracy.
robots.txt Review
 saroj.in/robots.txt Audit robots.txt to confirm no important collection or product URLs are disallowed. Add a Crawl-delay directive to reduce the 429 rate limiting. Ensure Screaming Frog and Googlebot are not blocked. Shopify's default robots.txt disallows /admin, /cart, /orders — these should remain disallowed.
Wholesale Page SEO
 pages/wholesale-fabrics The /pages/wholesale-fabrics page is missing an H1 and likely thin on content. This is a high-intent page for B2B buyers searching "wholesale fabric India" or "fabric wholesaler Surat/Mumbai". Add H1, meta description, 300+ words of content, and a contact/enquiry form with schema markup.
H2 Structure on Collection Pages
 All collection pages Once H1 tags are fixed, ensure each collection page also has a logical H2 structure within the description block. Use H2s for subsections like "About [Fabric Type]", "Available Designs", "How to Care for [Fabric]". This improves topical depth signals and supports voice search queries.

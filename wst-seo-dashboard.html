02 — Critical Issues

Issues Requiring Immediate Action

These 4 critical issues represent fundamental technical failures that directly limit search visibility, waste link equity, or send contradictory signals to Google. Each issue deducts 3 points from the health score — a total of −12 points.

CRITICAL

Issue Priority Scale Description How to Fix
Images Missing Alt Text
312 Webflow CDN images have no alt text across product, industry, and blog pages.
 Critical 312 images Every image on the Webflow site — product photos, industry application images, technical diagrams, and UI icons — lacks alt text. This fails WCAG 2.1 AA accessibility standards and prevents Google Image Search indexing. For a manufacturer, image search is a significant B2B discovery channel (engineers image-search product types). In Webflow Designer, select each image element → Settings panel → add descriptive alt text. Prioritise: product images (inflatable seals, silicone hoses, o-rings), the 'plus 2.png' icon (154 inlinks), Component 6.avif (24 inlinks), and all train-front images (68 inlinks each).
All Product Pages Missing Meta Descriptions
29 product pages have no meta description — every single product URL on the site.
 Critical 29 pages Without meta descriptions, Google auto-generates SERP snippets from page content — often poorly. For high-intent B2B searches like "inflatable seals manufacturer" or "silicone hose supplier", a crafted meta description with specs, certifications, and a CTA can dramatically improve click-through rate from engineers evaluating vendors. In Webflow, go to each product page → SEO Settings → write a 150–160 char meta description. Prioritise: o-rings, gaskets, inflatable seals, silicone hoses. Include material grade (FDA, EPDM, FKM), application (pharma, food, automotive), and a differentiator ("ISO 9001 certified", "custom specifications available").
Massive Redirect Link Equity Loss
/innovations-internal (816 inlinks) and /innovations (68 inlinks) both redirect to the homepage — 884 links wasted.
 Critical 884 inlinks These are almost certainly internal links left over from a deleted "Innovations" page or section. 816 internal links pointing to a redirect chain → homepage is the single largest link-equity waste in this audit. Every redirect loses some PageRank before passing it on. 1) Search all Webflow pages for links to "/innovations-internal" and "/innovations". 2) Either restore an Innovations page (case studies, R&D content) or update all 884 links to point to the most relevant existing page (/products or a specific product). Do not leave them pointing to the homepage.
www / non-www Canonical Conflict
8 pages canonical to www.westernsealtech.com but the server 301-redirects www → non-www. Direct contradiction.
 Critical 8 pages The canonical tag says "the authoritative version is www" but the server says "www doesn't exist, go to non-www." Googlebot receives contradictory signals: the canonical points one direction and the redirect points the other. This can cause indexing confusion for the homepage and key pages (contact, about, blog, careers). Choose one canonical version (recommend: non-www, since that's what the server serves). In Webflow Site Settings → SEO → ensure canonical URL uses non-www. Verify the Webflow project's custom domain settings are configured for non-www as primary. All canonical tags should then automatically align.

Opportunity Effort Impact Description Action
Product Schema (JSON-LD) Medium High No structured data found on product pages. Engineers increasingly use Google's AI Overviews and rich results to shortlist vendors. Product schema with material grade, application, and certifications can appear in these surfaces. Add Product JSON-LD schema to all product pages via Webflow custom code embed or a third-party schema app. Include: name, description, material, brand, and manufacturer fields.
FAQ Schema on Technical Blog Posts Low High Blog posts like "Why Inflatable Seals Fail Early" and "Inflatable Seals for Pharma Isolators" contain technical Q&A content that qualifies for FAQPage schema — giving accordion-style rich results in SERPs. Add FAQPage JSON-LD to blog posts containing question-answer structures. These posts rank well for engineer intent queries and rich results will significantly increase CTR.
Consolidate /product/ and /products/ URL Patterns Medium Medium The site uses both /product/ (singular) and /products/ (plural) as URL patterns. Custom-molded-seals, spliced-vulcanised-seals, rubber-extrusions, silicone-tubing-and-hoses, and diaphragms all use /product/ while others use /products/. Inconsistent URL structure confuses crawlers. Choose one pattern (recommend /products/) and 301-redirect all /product/ URLs to their /products/ equivalents. Update internal links to use the canonical pattern. This also fixes meta description coverage.
Blog-to-Product Internal Linking Low High The technical blog posts on pharma isolators, autoclave failures, and sandblasting seals are high-quality content but don't link aggressively to relevant product pages. This wastes the authority these posts are building. Add 2–3 contextual product links within each blog post body. "Inflatable Seals for Pharma Isolators" should link to /product/inflatable-seals. "12X Lifespan" case study should link to relevant product + Contact Us.
Add llms.txt for AI Search Visibility Low Medium Engineers increasingly use AI-powered search (Perplexity, ChatGPT, Google AI Overviews) to research seal specifications. An llms.txt file helps AI crawlers understand which pages represent the brand's core offerings. Create /llms.txt listing key product pages, about page, certifications page, and top blog posts. This takes <30 minutes and improves citability in AI-generated responses for seal-related queries.
Add Certifications & Compliance Page Medium High For pharma, food, and aerospace clients, ISO certifications, FDA compliance, and material grade certifications are primary vendor selection criteria — but a dedicated certifications page was not found in the crawl. This is a major EEAT (Expertise, Authority, Trust) gap. Create a /certifications or /quality page listing all ISO, FDA, RoHS, REACH, and industry certifications. Add schema markup (Organization) with hasCredential properties. Link from all product pages and the about page.
HrefLang for International Audiences Medium Medium Western Seal Tech serves clients in Netherlands (Breda), US (Wilmington, DE), and India (Mumbai) — confirmed by Google Maps links on the contact page. No hreflang tags were detected, meaning Google may not serve the site to the most relevant regional audience. If content is English-only, add hreflang="en" pointing to the canonical non-www URL. If regional pages are planned, implement hreflang with regional x-default fallback. Also verify Google Search Console is set to target the correct country.
